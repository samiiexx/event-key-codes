# Event Key Codes
A JavaScript Project from Brad Traversy's 50 Projects in 50 Days. This project is focused on building key codes. Clicking any key on your keyboard generates the key code. 

![Event Key Codes](screenshot.jpg)

## Built With
- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Install
```
    $ git clone git@github.com:samiiexx/sam-js-11.git
    $ cd main
```
